# StockPulse

StockPulse is a full-fledged stock trading and monitoring platform inspired by **Zerodha**. Built using the **MERN stack**, it offers real-time market monitoring, secure authentication, and seamless stock trading features. This project encompasses both **frontend and backend development**, including a **dashboard for buying and selling stocks**.
